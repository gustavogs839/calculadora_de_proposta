# Calculadora de Proposta

Projeto de calculadora de proposta comercial para simulação de financiamentos imobiliários.

## O que faz

- Permite inserir dados do empreendimento e unidade
- Calcula automaticamente entrada, valor de ato, HABITE-SE e valor a ser financiado
- Permite criar uma programação de parcelas com periodicidade e observações
- Gera proposta em PDF com layout mais profissional

## Como usar

1. Abra `index.html` em um navegador.
2. Preencha o valor do imóvel e porcentagens de entrada, imobiliária e construtora.
3. Adicione linhas de programação de parcelas e insira valores e observações.
4. Clique em `Calcular Proposta` para ver o resumo.
5. Clique em `Gerar PDF` para exportar a proposta.

## Estrutura do projeto

- `index.html` - interface do usuário
- `styles.css` - estilos da aplicação
- `script.js` - lógica de cálculo e geração de PDF
